array_diff_key - computa a difernça de arrays usando as chaves para comparação

array_diff_ukey - computa a diferença de arrays usando uma funçao callback nas chaves de comparação
array_diff - calcula a diferença entre arrays
array_fill - preenche um array com um determinado valor
array_filter - filtra elementos do array ultilizando a função de callback
array_flip - retorna um array com chaves e valores trocados
array_intersect - calcula a intersecção entre arrays
array_intersect_key - computa a intersecção de arrays com checagem adicional de indices por meio da função callback
